# 🏁 Shashka — 30 Davlat Qoidalari bilan

Ushbu loyiha brauzer orqali o'ynaladigan interaktiv shashka o'yini bo'lib, unda dunyoning 30 xil davlati (Rossiya, Braziliya, Kanada, Xalqaro va h.k.) qoidalari jamlangan. O'yinni do'stingiz bilan yoki aqlli botga qarshi o'ynashingiz mumkin.

## 🚀 Texnologiyalar
Loyihani yaratishda quyidagi texnologiyalardan foydalanilgan:
* **HTML5** — O'yinning strukturasi va asosiy elementlari uchun.
* **CSS3** — Zamonaviy dizayn, gradientlar, animatsiyalar va shisha (glassmorphism) effektlari uchun.
* **JavaScript (ES6+)** — O'yin mantiqi, qoidalar algoritmi va bot intellekti uchun.

## 🛠 O'rnatish va Ishga tushirish (VS Code orqali)

Agar siz ushbu kodni o'z kompyuteringizda tahrirlamoqchi yoki ishlatmoqchi bo'lsangiz, quyidagi bosqichlarni bajaring:

### 1. Visual Studio Code (VS Code) o'rnatish
1.  [code.visualstudio.com](https://code.visualstudio.com/) saytiga kiring.
2.  Operatsion tizimingizga mos (Windows, macOS yoki Linux) versiyasini yuklab oling.
3.  Yuklangan `.exe` yoki `.pkg` faylni oching va "Next" tugmalarini bosish orqali o'rnatishni yakunlang.
